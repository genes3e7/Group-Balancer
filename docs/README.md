# 🚀 Getting Started

Welcome to the **Group Balancer** documentation. This tool leverages Constraint Programming (CP-SAT) to solve complex group partitioning problems with high precision.

## Quick Start

1. **Upload Data:** Provide an Excel or CSV file containing participants and their score dimensions.
2. **Configure:** Define group capacities, score weights, and categorical constraints (`Sep`/`Grp`).
3. **Solve:** Run the optimizer and view live progress.
4. **Export:** Download the balanced results back to Excel.

## Key Features
* **Multi-Dimensional Scoring:** Balance groups across multiple numeric criteria.
* **Hard/Soft Categorical Constraints:** Enforce mandatory groupings or distribute types evenly.
* **Custom Capacities:** Specify exact sizes for every group.
* **Live Solver Feedback:** Monitor search progress and optimality gaps.
